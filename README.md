### ***How to run:***
**Simple make command that compiles all C programs**
```
make
```
**Compile OpenMP program**
```
make omp
```
**Compile POSIX Thread program**
```
make pt
```
**Run executables (after successful compile)**
```
# #.path = '#'
# OpenMP (replace #)
make omp#

# POSIX Thread (replace #)
make pt#
```
